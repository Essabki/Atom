# ⚛️ Atom 3D Animation

A simple **3D Atom visualization** built using **HTML, CSS, JavaScript, and Three.js**.  
This project demonstrates basic 3D rendering, animation, and orbital motion of electrons around a nucleus.

## 🌌 Preview

A 3D animated atom where electrons orbit around a central nucleus with smooth motion using WebGL via Three.js.

---

## 🚀 Features

- Real-time 3D rendering with Three.js
- Animated electron orbits
- Smooth rotation and camera movement
- Lightweight and beginner-friendly code
- Responsive canvas

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js (WebGL library)
